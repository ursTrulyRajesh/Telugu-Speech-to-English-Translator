# Telugu Speech to English Translator 

A simple web application that recognizes Telugu speech and provides real-time English translation.

### Features:
- **Speech-to-Text:** Uses Web Speech API for Telugu recognition.
- **Translation:** Integrated with MyMemory API.
- **Responsive UI:** Clean and simple design using HTML/CSS.

### How to use:
1. Open the project in Google Chrome.
2. Click "Start Listening" and allow microphone access.
3. Speak in Telugu and see the magic!
